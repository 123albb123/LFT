# 内网文件传输工具 · 使用说明

## 1. 安装与启动

本软件为免安装版本。下载 ZIP 后先完整解压，再运行 `内网文件传输工具.exe`。不要放在 `Program Files` 等通常需要管理员权限的目录中。

启动后软件会自动监听默认端口 `28080`。关闭主窗口后，传输服务随程序一起停止并释放端口。

Windows 防火墙首次询问时，如需让手机或其他电脑访问，请只勾选“专用网络”。软件不会自动添加或修改防火墙规则。

## 2. 从其他设备访问

确保两台设备连接到同一个路由器或局域网。在桌面端选择正确的本机 IPv4 地址，然后：

- 点击“显示二维码”，用手机扫码打开 Web 页面。
- 点击“打开 Web 页面”，在本机浏览器中打开管理页面。
- 手动输入界面中的地址，例如 `http://192.168.1.88:28080/web`。

文件也可以通过直链访问：

```text
http://192.168.1.88:28080/文件名
```

中文文件名会自动进行 URL 编码，复制链接后可直接发送给局域网中的其他设备。

## 3. 桌面端操作

- **添加文件**：点击“添加文件”，或把文件拖到共享文件区域。复制时会显示当前文件名、已复制大小、总大小和百分比，可随时取消。
- **更多操作**：右键单击文件，可复制文件链接、显示文件二维码、编辑、删除或打开文件目录。
- **启动/停止服务**：服务停止后，其他设备无法访问；重新启动后恢复访问。
- **日志**：简要记录服务状态，以及上传、下载、访问、打开编辑、实际文件变化和删除操作的来源。

共享文件默认按修改时间从新到旧排列。文件被外部程序修改、覆盖或删除后，列表会自动更新，原文件链接不变。

## 4. Web 页面操作

Web 页面会自动适配手机和电脑：

- 点击“选择文件”或拖放文件上传。
- 上传和下载过程中显示百分比。
- 上传完成后文件列表自动刷新。
- 点击文件名可直接访问，点击“下载”保存文件。
- 点击“复制链接”可分享直链。
- 只有桌面端开启“Web 删除”后，Web 页面才允许删除文件。
- 页面与服务连接异常时会保留错误提示，可点击“重新连接”同时恢复接口和进度事件连接。

## 5. 设置说明

- **共享目录**：默认为程序旁的 `Data`。可改为其他目录；修改后服务会受控重启。
- **端口**：允许 `1024–65535`，默认 `28080`。端口被占用时软件会提示。
- **同名处理**：支持覆盖、自动重命名和拒绝。
- **上传上限**：限制 Web 单个文件的最大大小，默认 2 GB。
- **仅允许局域网访问**：默认开启，仅允许本机和同一局域网客户端访问。
- **VPN/虚拟网卡**：Tailscale 等网卡会标记为虚拟/VPN，仍可手动选择；选择前请确认共享范围。
- **Web 上传/Web 删除**：可分别控制；Web 删除默认关闭。

设置、共享文件和日志默认保存在：

```text
Data/
Config/settings.json
Logs/
```

## 6. 常见问题

### 手机打不开页面

1. 确认手机和电脑在同一局域网，不要使用访客网络。
2. 在桌面端切换到与手机同网段的 IPv4 地址。
3. 确认服务状态为“服务运行中”。
4. 检查 Windows 防火墙是否允许该程序通过专用网络。
5. 检查路由器是否开启了 AP 隔离或客户端隔离。

### 提示端口被占用

在基础设置中更换一个高位端口，例如 `28081`，保存后重新访问新地址。

### 退出时仍有传输

软件会提示仍有传输任务。确认退出会中止传输并清理未完成的临时文件。

### 文件链接没有变化，但内容需要更新

直接覆盖或编辑共享目录中的同名文件即可。服务会更新缓存标识，浏览器重新访问时获得最新内容。

## 7. 安全建议

- 不要在公共 Wi-Fi 或不可信网络中运行。
- 不要分享密码、密钥、证书等敏感文件。
- 用完后关闭软件，服务和端口会随之停止。
- 如果不需要别人上传文件，可关闭“Web 上传”。
