# 更新记录

## 1.2.2

- 将应用提示框和二维码窗口统一为绿色圆角主题，完善中文操作按钮与键盘关闭行为。
- 修复运行日志内容垂直居中，统一为左上对齐并优化内边距。
- 关于窗口补充构建时间、真实绑定模式、监听地址和端口信息。
- 修正二维码标题重复，以及无活动传输清理时间仍为两小时的问题。

## 1.2.1

- 修复非法和零字节 Range 请求返回 500，并补全 HTTP 异常日志。
- 修复桌面最小窗口遮挡、周期性无效刷新、多选丢失及本地导入无进度问题。
- 完善 Web 断线恢复、下载失败状态、手机窄屏布局与任务清理。
- 支持 Tailscale `100.64/10` 直连网卡、IPv6 Scope ID URL，并改进端口恢复回滚。
- 精简重复操作入口，统一搜索提示、网卡完整提示和文本编辑行为。
